После билда игра не запустится, т.к. файлы, имеющие прописанный в коде путь не попадают в билд.
Для запуска необходимо скопировать папку ChunksInfo со всем её содержимым, находящаяся по пути Assets\Resources\
в билд по пути VoxelTest_Data\Resources\
После проделанной манипуляции игра будет работать.
